## Component Breakdown

### Left Side: Main Content (flex-1)
1. **TaskHeader** (sticky top)
   - Task icon + editable title
   - Close button (X)

2. **TaskMetadata** (2x2 grid)
   - Status pill
   - Assignee avatar
   - Date range picker
   - Budget display

3. **TaskDescription** (rich text editor)
   - Editable description
   - AI prompt bar (/) trigger

4. **TaskTabs** (tabbed content)
   - Details tab (custom fields)
   - Subtasks tab (checklist)
   - Action items tab (checklist)

### Right Side: Activity Sidebar (w-80, bg-gray-50)
1. **ActivityHeader** (border-bottom)
   - "Activity" title
   - 3 icon buttons (Search, Bell, More)

2. **ActivityTimeline** (flex-1, scrollable)
   - Activity items with timeline dots
   - User name + action + timestamp
   - "Show more" expansion
   - Empty state when no activities

3. **CommentInput** (border-top, sticky bottom)
   - Auto-resize textarea
   - Character counter
   - Toolbar with 7 icons
   - Comment type dropdown
